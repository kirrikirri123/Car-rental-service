Välkommen till Wigellkoncernens Styleguide

Syfte:

Denna sidan har syftet att guida dig i den grafiska profilen inom Wigellkoncernen.
Målet är en enkel, snabb lathund till implementeringen i nya och befintliga element.

How-to:

Varje område är indelat i tre boxar med olika syften.

Inom den markerade ramen så finns det grafiska elementet. Dock ligger paneler och meddelanden utanför exempel-ram för ökad tydlighet.

Inom den tunnare ramen finns beskrivande text av grafiska element och stilval.

Inom den sträckade ramen finns kodimplementationen iform av CSS och HTML redo att kopieras.
Knapp med CSS-symbol kopierar CSS och knapp med HTML-symbol kopierar html.

Paneler

För paneler används enkla rektangulära block med standard bakgrundsfärgensnyans med en opacitet på 0.89.
Rubriker i storleken h3 och infotext i standard storlek.
Runt panel används en tunn ram av standard grånyans.
Använd en inre div för att skapa en ram-effekt inom panel-elementet.
För dem olika typerna av paneler används olika ramar och hoovrings effekter. På touch blir effekten vid tryck.
Neutral = En dashed ram med standardnyans som ligger endast ovanpå. Effekten är att ramdelen blir även till vänster och solid.
Postiv= En solid ram med gul nyans som ligger som ligger endast ovanpå. Effekten är att ramdelen även blir nedtill samt dashed.
Negativ = En solid ram med röd/orange nyans som ligger endast nedtill. Effekten är att det blir en full ram runt objektet.

Meddelanden & Varningar

Bekräftande

Lyckad åtgärd!

Info om det positiva som lyckades!

Bekräftande meddelande

Vid lyckad sparning eller annan icke negativ händelse används en enkel meddelanderuta som bekräftelse för att visa händelsen. Meddelanderutan visas antingen genom förändring i ett fast element eller som popup. Bakgrunden är neutral men markerar sitt positiva budskap med text,symbol och färgen gul. 


<div class="message">
    <div class="message-pos">
        <div class="message-text">
            <h3>Beskrivande rubrik</h3>
            Info om vad som lyckades.
        </div>
        <div id="icon-pos"><i class="bi bi-hand-thumbs-up"></i></div>
    </div>
</div>
            

                            
/* Yttre meddelande-rutans div */
.message{
    width:clamp(20rem,55%,34rem);
    border:0.05em solid var(--border-color);
    background-color: var(--bg);
    padding:1em;
}
/* Textens div */
.message-pos{ 
    display: flex;
    flex-direction: row;
    justify-content:space-evenly;
    border-bottom:0.2rem dashed #FFEB3B;
    padding:0.5em;

/* Ikonens div */
#icon-pos{

}
/* Klass för marginaler mellan text och ikon. */
.message-text{
    margin-right: 1.2rem;
}

Varnande

Åtgärd avbruten!

Info om vad som inte lyckades.

Varnande meddelande

Vid misslyckad sparning eller annat negativt event används en enkel meddelanderuta som varning att visa att något inte gick enligt plan. Visas genom förändring antingen i ett fast element eller popup. Bakgrunden är neutral men markerar sitt varnande budskap med text,symbol och en rödaktig färg.

    </div>
</div>

Laddningsindikator

Laddningsindikatorn är ett snurrande hjul som att visa på en aktiv process. Går att sätta i grön #5A824F, grå #878672 eller mörkare #292919


/*Animeringen av snurr*/
@keyframes spin{
    from {transform: rotate(0deg);}
    to {transform: rotate(360deg);}
}
/* Ikon utan färg */
.loading-icon{
    animation: spin 5s linear infinite;
    display: inline-block;
    font-size: 2rem;
}
